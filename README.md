# Consultora Concesionarias — Dashboard Multi-Cliente

Dashboard centralizado para visualizar KPIs de marketing digital de múltiples clientes.

## Stack

- **Frontend**: Next.js 15 + TypeScript + Tailwind + shadcn/ui + Recharts — Vercel
- **Backend**: Fastify + Node.js + TypeScript — VPS propio (pm2 + Caddy)
- **DB**: MySQL + Prisma ORM
- **Auth**: NextAuth.js v5
- **Cron**: node-cron (8 AM Argentina)
- **IA**: Anthropic Claude API

## Estructura

```
frontend/   → Next.js (Vercel)
backend/    → Fastify + cron (VPS)
docs/       → Documentación interna
```
